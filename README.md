# 🌟 Samarpan NGO Management System

A full-stack MERN application designed to streamline NGO operations by providing dedicated portals for **Admins, Donors, and Volunteers**. The platform simplifies donation management, campaign organization, volunteer coordination, and event management through a secure and user-friendly interface.

## 🚀 Features

### 👨‍💼 Admin Portal

* Manage campaigns, events, blogs, and volunteers
* Track donations and generate reports
* Dashboard with analytics

### 💝 Donor Portal

* Secure online donations via Razorpay
* Track donation history
* View campaigns and download receipts

### 🙋 Volunteer Portal

* Register for volunteering opportunities
* Apply for events
* Track participation and updates

## 🛠 Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* React Router

### Backend

* Node.js
* Express.js
* MongoDB

### Tools & Integrations

* JWT Authentication
* Razorpay Payment Gateway
* Cloudinary
* Nodemailer

## ✨ Key Features

* 🔐 Role-Based Authentication (Admin, Donor & Volunteer)
* 💳 Secure Online Donation System

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/janakgadhiya/Samarpan-NGO.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend/ngo
npm install
npm run dev
```
